## 내린 결정
- **실키 봇 e2e를 구독 경로로 실제 수행**(API 키 불필요):
  `claude-agent-sdk` 설치 → `CliAgentRunner`로 실제 모델 호출.
  1) 인사 턴 실응답 OK. 2) 툴유즈: 봇이 present_place_options 호출→옵션 3개 카드 emit OK.
  3) **풀스택**: 실 uvicorn + 구독 봇 + 실 WS로 채팅→@봇→툴→카드→WS 브로드캐스트 종단.
  → `tests/test_cli_e2e.py` 영구 테스트로 고정. **RUN_CLI_E2E=1 실측 1 passed(68s)**.
  기본·CI는 skip(claude CLI/SDK·플래그 없을 때) — CI 그린 유지.
- **CI를 GitHub 실제 러너에서 원격 검증**(로컬 그린을 넘어):
  CI run 성공 — 백엔드 pytest ✓(15s), 프론트 빌드+SSR ✓(14s).
  Deploy run 성공 — 시크릿 없어 guard에서 Fly·CF 잡 깔끔히 건너뜀(3~4s).

## 남은 작업 (자격증명/계정 필요 — 내가 대신 불가)
1. [자격증명] Kakao 실검색/실동선 — KAKAO_REST_API_KEY 필요(없으면 봇이 추정값으로 카드 채움).
2. [자격증명] 실제 프로덕션 배포 — GitHub Secrets에 FLY_API_TOKEN / CLOUDFLARE_API_TOKEN /
   CLOUDFLARE_ACCOUNT_ID 넣고 main 푸시(또는 로컬 flyctl/wrangler). 파이프라인은 이미 원격 검증됨.
3. [폴리시] 룸 셸/지도 실연동(Kakao Map)/모바일 바텀시트 픽셀 파리티 — 기능 블로커 아님.
