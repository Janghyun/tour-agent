모바일을 확인하는 다른 방법

  1. 이 페이지 그대로 — 위 폰 프레임에서 바로 조작
  2. 앱을 새 탭/창으로 열고 브라우저 창 너비를 760px 이하로 줄이기
  3. 크롬/사파리 개발자도구 → 기기 모드(Device Toolbar)에서 iPhone 선택
  4. 실제 휴대폰에서 열기 (배포 후)
